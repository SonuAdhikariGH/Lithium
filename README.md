Lithium ⚛️

A minimalist, frictionless, and completely anonymous micro-blogging platform. No sign-ups, no logins, no profiles—just pure, unfiltered expression. Think of it as a stripped-back, privacy-first mini Twitter where thoughts flow instantly.
✨ Features

    Zero Friction: No account creation or login required. Open the app and start reading or posting immediately.

    Total Anonymity: No identities, handles, or avatars. Every post stands solely on the weight of its content.

    Real-time Feed: A unified public timeline where thoughts, ideas, and micro-updates from around the world appear instantly.

    Minimalist UI: A clean, focused, and distraction-free design crafted for effortless readability.

📦 Installation & Setup

Get a local copy of Lithium up and running in just a few steps.
Prerequisites

Make sure you have Node.js and npm installed on your machine.
Steps

    Clone the repository:
    Bash

    git clone https://github.com/SonuAdhikariGH/lithium.git
    cd lithium

    Install dependencies:
    Bash

    npm install

    Start the development server:
    Bash

    npm start
